Barricora v44 Daily Checklists

Built from v43 Action Update Flow.

New in v44:
- Daily Checklists module added.
- General Daily Walk checklist added.
- Simple header: date, site/project, optional area, checked by.
- Simple Yes / No / N/A questions for PPE, SPA/RAMS, permits, work at height, barriers, housekeeping, tools, manual handling, plant interface, emergency access and unsafe behaviour.
- Multiple checklist photos supported.
- A No answer creates a checklist action.
- Daily checklist actions appear in the main Action / Issue List.
- Action Update / In Progress / Close-out workflow works for checklist actions.
- Checklist history stays saved as proof.
- Cloud API added: /api/daily
- New D1 schema file: schema_v44_daily_checklists.sql

After uploading to GitHub, run schema_v44_daily_checklists.sql in D1 before testing cloud daily checks.
